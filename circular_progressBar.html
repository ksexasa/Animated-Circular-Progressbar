Case where percentage is x > 50%


1. From percentage X substract 50
2. Multiply the result by 3.6
3. Add it to the degrees of the 100% keyframe
Formula (X - 50)*3.6
For example the percentage 83%: (83 - 50) * 3.6 => 33 * 3.6 => 118.8

Case where percentage is x < 50%


1. Multiply the percentage X by 3.6
3. Add it to the degrees of the 100% keyframe
Formula X*3.6
For example the percentage 7%: 7 * 3.6 => 25.2

Note: The animation starts from 50 percent in the first case. I want the code to have the minimum amount of lines so I will keep it this way unless there is a clever way to fix this.
